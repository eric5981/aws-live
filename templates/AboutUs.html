About Us

We specialize in providing Human Resource Management System, with our main focus being on employee management information. Our system goals is to manage the employee details, attendance of employee and payroll of employee in the system. The system is runnng by Amazon Web Services (AWS) which consist of server of Linux created by Amazon Elastic Compute Cloud (EC2), storage of image in Amazon Simple Storage Service (Amazon S3), data stored in Amazon Relational Database Service (Amazon RDS), Amazon CloudWatch used to monitor the AWS resources and system real-time, Amazon Auto Scaling to automatic monitor and adjust the compute resources to maintain the performance of system in good condition and Elastic Load Balancing to automatic distribute incoming application traffic from the instances that the system running.
